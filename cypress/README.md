# E2E тесты Cypress

## Установка

```bash
npm install
npm run cypress:install
```

`npm install` ставит Cypress и вспомогательные dev-зависимости. Команда `npm run cypress:install` нужна только если бинарник Cypress не скачался автоматически или кеш был очищен.

## Запуск

```bash
npm run e2e
```

Команда сама поднимает приложение на `http://localhost:3000`, дожидается готовности сервера и запускает все тесты из `cypress/e2e`.

Для интерактивного режима:

```bash
npm run e2e:open
```

## Что покрыто

- `library.cy.js` проверяет открытие библиотеки, наличие книг и фильтр по жанру.
- `bookPage.cy.js` проверяет переход из библиотеки на страницу книги и возврат обратно.
- `readerModal.cy.js` проверяет открытие `ReaderModal`, листание страниц и закрытие модального окна.
